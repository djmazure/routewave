# RouteWave

FPGA Waveform Viewer — release binaries.

## Download

Pre-built binaries are published as [GitHub Releases](https://github.com/djmazure/routewave/releases).

Supported platforms:
- **Linux x86_64** — tarball + AppImage
- **Windows x86_64** — zip (CLI only, cross-compiled)

## Install via RouteRTL SDK

```bash
pip install routertl
rr ws update-viewer    # auto-downloads the latest RouteWave binary
rr waves my_sim.fst    # opens the waveform viewer
```
